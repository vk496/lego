LEGO
===

# Start the project

For virtual network
```bash
$ git clone -b sta_tci https://github.com/vk496/lego
$ cd lego
$ ./up.sh -b
```

For external attached network
```bash
$ git clone -b sta_tci https://github.com/vk496/lego
$ cd lego
$ ./up.sh -m iface1 -m iface2
```
